Lista de Exercícios 09 - Módulos, npm e webpack

1. Escreva um módulo do Node.js que exporte uma função qualquer que receba um ou mais parâmetros 
e retorne um valor (você pode reaproveitar uma das funções escritas em listas anteriores, por exemplo).

Escreva o código principal em outro arquivo, fazendo uma chamada à função definida no módulo 
e escrevendo a saída no console.

2. Reescreva o módulo anterior com a sintaxe do módulo nativo e teste-o em um navegador, 
integrado a uma página HTML (faça a chamada da função em um arquivo .js diferente daquele onde a função foi definida).

3. Utilize o npm para baixar o módulo to-space-case 
(https://www.npmjs.com/package/to-space-case). 
Escreva na saída o resultado da chamada dessa função para as strings:

    InstitutoFederalDeBrasilia
    sistemas.para.internet
    Programacao_para_Internet-1

4. Utilize o arquivo package.json abaixo para instalar os pacotes adicionais listados nas dependências (sem digitar os nomes desses pacotes na linha de comando):

{
    "name": "lista10",
    "version": "1.0.0",
    "description": "Programação para Internet I",
    "dependencies": {
        "to-capital-case": "^1.0.0",
        "to-constant-case": "^1.0.0",
        "to-space-case": "^1.0.0"
    }
}

Teste os novos módulos instalados (to-capital-case e to-constant-case) com as mesmas strings da questão anterior.

5. Usando o Webpack, adapte o código anterior para funcionar no navegador. 
Faça uma página com uma caixa de texto e um botão. 
Ao pressionar o botão, deve ser exibido ao usuário o resultado da aplicação 
de uma das funções acima no texto digitado.

6. Divida o código do seu projeto em ao menos 2 arquivos .js diferentes, 
fazendo em um referência ao outro (o resultado final ainda deve ser um único arquivo .js gerado pelo Webpack).
